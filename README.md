Name: Star View Cinema

Description:
StarViewCinema is a web-based application designed to streamline the movie ticket
reservation process for users while providing cinema administrators with powerful management tools.
The system enables secure user registration, seat selection, ticket booking, and payment,
while also offering an efficient way to manage movie schedules, pricing, and user feedback.

Objectives:

1- To provide a user-friendly platform for booking movie tickets online.
2- To enable secure authentication for users and administrators.
3- To offer an interactive seat selection and multiple payment options.
4- To enhance customer satisfaction with discounts, rewards, and feedback collection.
5- To support cinema staff with efficient management tools for movies, users, and ticket pricing.
